# Hospital Management System (SQL Project)

## Project Description
Hospital Management System is a database project designed using SQL to manage hospital operations including patients, doctors, appointments, and billing with structured data handling and reporting.

## Technology Used
- MySQL

## Database Tables
- Doctors: Stores doctor details and specialization  
- Patients: Stores patient information  
- Appointments: Manages doctor-patient scheduling  
- Bills: Stores billing records and payment details  

## SQL Concepts Used
- DDL (CREATE, ALTER, DROP)  
- DML (INSERT, UPDATE, DELETE)  
- Primary Key & Foreign Key  
- Joins (INNER JOIN)  
- Aggregate Functions (SUM, AVG, COUNT, MIN, MAX)  
- GROUP BY  
- HAVING  
- Subqueries  
- ORDER BY  
- LIMIT  
- Views
